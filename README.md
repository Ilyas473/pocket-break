# Pocket Break - Smart Vending Cognitive Interface

A production-ready research prototype for an interactive vending machine that gamifies healthy snack selection through adaptive cognitive challenges.

## Installation

```bash
npm install
npm start
```

## Features

- Interactive 3D vending machine simulation
- Adaptive cognitive game engines (Card Sorting, Distractor Game)
- Real-time cognitive profile tracking
- Reanimated 3 hardware physics simulation
- Zustand state management

## Architecture

- `src/store/` - Centralized state management
- `src/features/vending/` - Hardware simulation
- `src/features/games/` - Cognitive game engines
- `src/components/` - Shared UI components
- `src/config/` - Theme and styling
